Listas de TAGs HTML

Abaixo deixarei algumas de minhas TAGs prediletas salvas

  1. Títulos: <h>
  2. Para marcações no HTML com essa: <abbr>
  3. Parágrafo: <p>
  4. Para colocar links internos e externos: <a>
  5. Para inserir imagem: <img src>
  6. Para listas ordenadas ou desordenadas: <ol>, <ul>, <li>
  7. Para pular uma linha ou fazer um parágrafo: <br>
  8. Para fazer uma linha horizontal: <hr>
  9. Antes de escrever códigos para deixar a letra igual: <code>
  10. Para escrever um texto aleatório com quantidades N de palavras: <loremN>
  11. Para respeitar os espaços e enter do código ou texto: <pre>
  12. Para citar um texto: <q>
  13. Para criar uma definição na lista: <dt>, <dd>
